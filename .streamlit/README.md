# Streamlit Configuration

This directory contains Streamlit configuration files.

## Files

- `config.toml` - Streamlit theme and server configuration
- `secrets.toml` - Local secrets (NOT committed to git - see .gitignore)
- `secrets.toml.example` - Example secrets file (safe to commit)

## Important

- `secrets.toml` is in `.gitignore` - never commit real secrets!
- `packages.txt` should NOT exist - system packages are not needed
- All Python packages are in `requirements.txt` in the root directory
